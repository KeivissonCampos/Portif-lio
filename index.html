Conheça um dos projetos Robot IA

Um projeto que transforma o celular em um robô inteligente, capaz de executar comandos personalizados e com respostas interativas utilizando inteligência artificial

O robô, construído em MDF, utiliza um celular para simular seus olhos por meio de um aplicativo desenvolvido em uma plataforma Low Code. Esse aplicativo se comunica com o circuito do robô via Bluetooth 4.0, permitindo que os movimentos sejam acompanhados pelos olhos do robô. O projeto integra diversas APIs, incluindo reconhecimento facial, conversão de texto em fala (Text-to-Speech), OpenAI, Meta AI, e Google Sheets, além de utilizar o protocolo UDP para comunicação com um ESP32.

Um site desenvolvido em uma plataforma No Code (Bubble.io) para empresas que buscam modernizar sua gestão de Recursos Humanos e Tarefas. Atualmente, estou aplicando os conhecimentos adquiridos no curso de Full stack, para reproduzir essa plataforma e aprimorando ainda mais suas funcionalidades.

Este robô combina peças de MDF e componentes impressos em 3D, com funcionamento baseado em um software desenvolvido em C# na plataforma Visual Studio. O software permite controlar os 13 servomotores, converter texto em áudio e simular lip sync no robô, proporcionando uma experiência interativa e realista.

Este site foi desenvolvido utilizando HTML para a estrutura, CSS para o estilo e JavaScript para a programação. Para a geração dos QR Codes, foi integrada uma API específica, permitindo criar códigos personalizados de forma rápida e eficiente.
